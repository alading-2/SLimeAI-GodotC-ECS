# 对象池管理器可视化测试 (C#)

这是一个升级版的、基于 C# 的对象池测试场景，专门用于测试 `ObjectPoolManager` 的全局管理能力和多对象池的协同工作。

## 🌟 核心特性

1.  **全局管理器监控 (ObjectPoolManager Stats)**

    - 实时展示 **所有** 注册对象池的状态（不仅是单个池）。
    - 监控每个池的闲置数、活跃数、创建总量、回收总量及复用率。
    - 验证 `ObjectPoolManager` 的单例管理和统计聚合功能。

2.  **多类型对象池测试**

    - **投射物池 (Projectile Pool)**: 模拟高频发射的子弹（绿色），具备物理反弹和长生命周期。
    - **特效池 (Effect Pool)**: 模拟短生命周期的视觉特效（洋红色），自动播放动画并销毁。
    - 验证不同配置、不同类型的对象池在同一场景下的共存与隔离。

3.  **交互式控制面板**

    - **独立控制**: 分别控制投射物和特效的生成/自动生成。
    - **全局操作**:
      - `清理闲置`: 测试 `ObjectPoolManager.CleanupAll()`，一键释放所有池的冗余内存。
      - `销毁所有`: 测试 `ObjectPoolManager.DestroyAll()`，验证全局重置能力。

4.  **智能对象表现**
    - **投射物**: 显示复用计数，颜色随复用次数变化。
    - **特效**: 播放缩放淡出动画，生命周期结束自动归还。

## 🚀 如何运行

1.  在 Godot 编辑器中打开项目。
2.  导航到文件系统：`res://Src/ECS/Test/SingleTest/Tools/ObjectPool/`。
3.  右键点击 `ObjectPoolManagerTest.tscn` 并选择 **运行场景 (Run Scene)** (或按 F6)。

## 🎮 操作指南

- **鼠标左键点击**: 在点击位置生成 **投射物**。
- **鼠标右键点击**: 在点击位置生成 **特效**。
- **左侧面板**:
  - 查看实时统计数据。
  - 开启/关闭各池的自动生成。
  - 执行全局清理或销毁操作。
