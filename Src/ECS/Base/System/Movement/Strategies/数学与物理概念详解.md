# 数学与物理概念详解

## 1. 基础运动学概念

### 速度与位移

```csharp
Vector2 forwardDisp = _baseDirection * (_baseSpeed * delta);  // 位移 = 速度 × 时间
data.Set(DataKey.Velocity, totalDisp / Mathf.Max(delta, 0.001f)); // 速度 = 位移 / 时间
```

**物理原理：**
- 速度：矢量，有大小和方向
- 位移：矢量，位置变化量
- 关系：`v = Δs / Δt`，`Δs = v × Δt`

### 垂直向量计算

```csharp
Vector2 perp = new Vector2(-_baseDirection.Y, _baseDirection.X);
```

**数学原理：** 2D垂直向量
- 对于向量 (x, y)，其垂直向量为 (-y, x) 或 (y, -x)
- 这是逆时针旋转90度的结果
- 点积为零：`(x, y) · (-y, x) = -xy + xy = 0`

## 2. 正弦波动数学

### 正弦函数参数

```csharp
float sineOffset = WaveMath.EvaluateSine(
    @params.WaveAmplitude,
    @params.WaveFrequency,
    @params.ElapsedTime + delta,
    @params.WavePhase);
```

**标准正弦函数：** `y = A × sin(2π × f × t + φ)`

**统一封装：**
- `WaveMath.EvaluateSine(...)`：计算某个时刻的正弦偏移值
- `WaveMath.EvaluateSineDelta(...)`：计算两个时刻之间的正弦偏移差值
- 对外 `WavePhase` 继续使用“度”，工具内部自动转弧度

**参数解析：**
- A (振幅)：WaveAmplitude，最大偏离距离（像素）
- f (频率)：WaveFrequency，每秒波动次数（周期/秒）
- t (时间)：ElapsedTime，运动经过的时间（秒）
- φ (相位)：WavePhase，初始偏移角度（度）
- 2π：Mathf.Tau，完整圆周角，用于频率到角速度转换

### 增量计算法

```csharp
float sineDelta = WaveMath.EvaluateSineDelta(
    @params.WaveAmplitude,
    @params.WaveFrequency,
    @params.ElapsedTime,
    @params.ElapsedTime + delta,
    @params.WavePhase);
Vector2 sideDisp = perp * sineDelta;
```

**数学优势：**
- 避免累积误差：不直接累加位移
- 数值稳定：使用瞬时差分而非积分
- 导数关系：Δy/Δt ≈ dy/dt = A × 2π × f × cos(2π × f × t + φ)

### 正弦波朝向

```csharp
Vector2 tangentDirection = WaveMath.EvaluateSineTangent(
    _baseDirection,
    _baseSpeed,
    @params.WaveAmplitude,
    @params.WaveFrequency,
    @params.ElapsedTime + delta,
    @params.WavePhase);
```

**朝向语义：**
- `Velocity` 仍用于把实体推到下一采样点
- `FacingDirection` 应取轨迹切线 `forward * forwardSpeed + side * sine'(t)`
- 这样视觉朝向反映“曲线瞬时前进方向”，而不是“当前位置到目标采样点的纠偏方向”

## 3. 角度与弧度转换

### 角度系统

```csharp
float sineValue = WaveMath.EvaluateSine(amplitude, frequency, time, phaseDegrees);
```

这里仍然遵循项目统一约定：外部输入角度一律使用“度”，仅在 `WaveMath` 内部做 `DegToRad` 转换。

**Godot 2D 角度约定：**
- 0°：向右 (Vector2.Right)
- 90°：向下（因为 Y 轴向下）
- 180°：向左
- 正值：顺时针旋转

**转换公式：**
- 弧度 = 角度 × π / 180

## 4. 回旋镖运动学

回旋镖是依靠持续的侧向偏移形成回旋轨迹。当前实现没有模拟真实空气动力学，而是用更稳定、更可控的参数化近似。

### 双半椭圆采样

回旋镖的轨迹可以理解为一个局部坐标系里的半椭圆采样：

```csharp
P(progress) = Center + Forward * X(progress) + Side * Y(progress)
```

- `Forward` 表示起点到终点的主方向
- `Side` 表示与主方向垂直的侧向轴
- `X(progress)` 表示沿弦线推进
- `Y(progress)` 表示半椭圆弧线的鼓起高度

去程和返程使用同一种局部曲线模板，但返程会实时重建终点：

```csharp
// 去程：LaunchPoint -> TargetPoint
// 返程：折返点 -> TargetNode 当前坐标；若宿主失效则回退 LaunchPoint
```

这样做的结果是：
- 去程和返程都会明显“兜弯”
- 返程不会死盯发射时旧坐标
- 轨迹形状由 `BoomerangArcHeight` 控制，方向由 `BoomerangIsClockwise` 控制

### 半椭圆弧长近似

为了让采样更接近匀速，策略不会按固定进度硬推，而是先估算半椭圆长度，再把 `speed * delta` 换算成进度增量：

```csharp
progressDelta = speed * delta / halfEllipseLength
```

半椭圆长度在实现里使用椭圆周长近似公式求得。这样比逐帧数值积分更轻，足够支撑投射物类运动。

### 切线朝向

回旋镖的视觉朝向不应该直接取当前位置到终点的向量，因为那样会在弧线末端产生“猛地朝目标拐头”的感觉。当前实现改为取曲线切线：

```csharp
tangent = Forward * dX + Side * dY
```

这会让朝向更像“顺着弧线甩出去又兜回来”，也更接近玩家对回旋镖的直觉预期。

## 5. 环绕运动物理

### 角运动参数

```csharp
public float OrbitAngularSpeed { get; init; } = 0f;        // 角速度（度/秒）
public float OrbitAngularAcceleration { get; init; } = 0f; // 角加速度（度/秒²）
public float OrbitTotalAngle { get; init; } = -1f;         // 总角度（度）
```

**物理概念：**
- 角速度 ω：ω = dθ/dt，单位时间转过的角度
- 角加速度 α：α = dω/dt，角速度的变化率
- 向心加速度：a_c = ω² × r，指向圆心
- 线速度：v = ω × r，切向速度

### 径向运动

```csharp
public ScalarDriverParams? OrbitRadiusScalarDriver { get; init; } = null;  // 半径动态驱动；null = 不驱动
```

螺旋运动：角运动 + 半径驱动的合成

```csharp
Vector2 radialDirection = new Vector2(cos, sin);
Vector2 tangentialDirection = new Vector2(-sin, cos) * sign;
Vector2 facingDirection = tangentialDirection * (radius * angularSpeedRad)
    + radialDirection * radialSpeed;
```

**朝向语义：**
- `Velocity` 可能是“当前位置被拉回目标轨道点”的纠偏速度
- `FacingDirection` 应取解析轨迹切线：切向速度 + 径向速度
- 当半径驱动产生非零 `ValueDelta` 时，视觉上就是螺旋切线，而不是纯圆周切线

### 通用边界驱动

`ScalarDriverParams` 用于描述“某个标量在区间中如何变化”：

- `Velocity`：每秒变化量
- `Acceleration`：每秒速度变化量
- `Min / Max`：上下界
- `MinResponse / MaxResponse`：触边后的响应方式

在 `MovementParams` 中推荐配合可空驱动字段使用：

- `null` = 不启用驱动，保持基础值常量
- 非 `null` = 由策略实例私有持有 `ScalarDriverState` 并逐帧调用 `ScalarDriver.Step()`

更完整的职责说明见 `../ScalarDriver/README.md`。

其中 `PingPong` 支持：

- `BounceDecay`：每次触边后速度衰减系数
- `StopSpeedThreshold`：速度衰减到阈值以下时冻结在边界

## 8. 坐标系统与变换

### Godot 2D 坐标系
- 原点：左上角
- X轴：向右为正
- Y轴：向下为正
- 旋转：顺时针为正

### 向量运算

```csharp
_baseDirection.Normalized()  // 单位向量
totalDisp.Length()           // 欧几里得范数
```

**线性代数基础：**
- 单位向量：长度为1的方向向量
- 范数：向量的长度（模）
- 点积：a · b = |a| × |b| × cos(θ)
