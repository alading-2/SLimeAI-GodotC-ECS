# 移动策略数学参考

Movement 执行规则见 `DocsAI/Modules/Movement.md`，策略设计见 `Docs/框架/ECS/System/Movement/移动系统设计说明.md`。

## 坐标与角度

- Godot 2D：X 向右，Y 向下
- 对外角度统一使用“度”，语义：`0=右、90=下、180=左、正值顺时针`
- 策略内部仅在三角函数或 Godot API 需要时转为弧度

## 常用公式

- 方向单位化：`Vector2.Normalized()`
- 垂直方向：根据策略选择顺/逆时针垂线
- 正弦波：基础方向前进 + 垂直方向振荡
- 贝塞尔：控制点曲线采样，追踪模式优先用模板重建剩余曲线
- 抛物线/圆弧：由起点、终点和形状参数生成单段轨迹
- 环绕：角运动 + 半径驱动合成
