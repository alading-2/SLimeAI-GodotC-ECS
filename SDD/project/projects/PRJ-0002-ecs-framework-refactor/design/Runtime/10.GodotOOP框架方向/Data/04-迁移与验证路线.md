## Grep gate 草案

后续代码切片可使用这些 gate：

```bash
rg -n "Data\\.Get<|Data\\.Set<" Src/ECS/Capabilities/*/Component -g '*.cs'
rg -n "SetUntyped|TrySetUntyped|GetAll" Src/ECS -g '*.cs'
rg -n "new DataModifier" Src/ECS Data -g '*.cs'
rg -n "SharedState|Shared State" DocsAI SDD/project/projects/PRJ-0002-ecs-framework-refactor .ai-config/skills -g '*.md'
```

这些 gate 不是为了全部清零，而是为了分类：

- Component 热路径是否还在散点读 Data。
- untyped Data API 是否只留在 loader/debug/test 边界。
- DataModifier 是否只用于 numeric attribute 字段。
- 文档是否又把 Data 改名成 SharedState。
