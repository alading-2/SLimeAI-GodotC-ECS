# 字符串插值即时求值问题

> 严重度：🟡 MEDIUM
> 影响范围：全局日志调用

## 问题描述

C#的字符串插值 `$"..."` 在调用时立即求值，即使日志级别被禁用。每次插值都分配一个新的string对象，值类型参数被装箱为params object[]。

## SlimeAI中的模式

```csharp
// 每次调用都分配字符串，即使Info级别被禁用
Log.LogInfo($"[AbilitySystem] {abilityId} triggered on {targetId}");
Log.LogInfo($"[DamageService] Processing {info.Amount} damage to {victim}");
Log.LogDebug($"[DataSlot] {key} modified to {newValue}");
```

**估算：** 假设每帧20次日志调用，每次平均15字符 + 2个装箱参数 ≈ 80字节/帧

## 解决方案

### 条件日志

```csharp
if (Log.IsInfoEnabled)
    Log.LogInfo($"[AbilitySystem] {abilityId} triggered");
```

### LoggerMessage源生成（零分配）

```csharp
[LoggerMessage(Level = LogLevel.Information, Message = "[AbilitySystem] {AbilityId} triggered")]
public static partial void AbilityTriggered(this ILogger logger, string abilityId);
```
