SystemAgent Optimization 项目用于统一管理 SystemAgent 与 SDD 体系升级：将共享设计资料收拢到项目级 `design/`，将 SDD 系统相关历史任务按顺序迁入 `sdds/`，并通过后续有序子 SDD 推进 CLI、workflow、skill、hook 和文档治理改造。
