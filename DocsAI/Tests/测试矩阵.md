## 批量回归

核心或跨模块修改后运行：

```bash
node .codex/skills/godot-scene-test/scripts/godot-scene-runner.mjs run-all --build --continue-on-fail
```

如果批量测试成本太高，至少运行相关模块测试和 `GlobalTest/MainTest`。
